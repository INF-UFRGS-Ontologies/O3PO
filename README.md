## Versions

### Stable release versions

The latest version of the ontology can always be found at:

https://www.inf.ufrgs.br/ontologies/o3po.owl

### Previous versions

The source files for O3PO were previously mantained in the repository [BDI-UFRGS/O3POntology](https://github.com/BDI-UFRGS/O3POntology/). Current and future versions are *not* backward compatible with the former versions available in that repository, mainly due to updated IRIs and entity identifiers.
