# Offshore Petroleum Production Plant Ontology (O3PO)

Description: A domain ontology for offshore petroleum production plants has been established utilizing BFO as the top-level ontology and incorporating IOF-Core and GeoCore as middle-level ontologies.


## Versions

### Stable release versions

The latest version of the ontology can always be found at:

https://www.inf.ufrgs.br/ontologies/o3po.owl


### Editors' version

Editors of this ontology should use the edit version, [src/ontology/o3po-edit.owl](src/ontology/o3po-edit.owl)

## Contact

Please use this GitHub repository's [Issue tracker](https://github.com/INF-UFRGS-Ontologies/O3PO/issues) to request new terms/classes or report errors or specific concerns related to the ontology.
